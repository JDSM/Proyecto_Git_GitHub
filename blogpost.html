Proyecto del Curzo Platzi Profecional Git y GitHub

Este proyecto contiene los comandos de uso de Git y GitHub.

Vamos a crear un proyecto con control de versiones usando Git, como subir nuevos cambios o versiones de nuestro proyecto. Movernos entre las versiones, para regresar a un estado anterior. Crear Ramas dentro de nuestro proyecto para que no interfieran con el desarrollo principal del proyecto. Unir las ramas al proyecto principal.

Se simulan conflictos entre ramas, para corregir los errores que se pueden presentar durante la union de diferentes ramas.
